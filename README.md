## Why Rust?

- [x] Concurrent
- [x] Fast by default
- [x] Similar to C/C++ languages
- [x] Multiple use case language: Web, WebAssembly, Games, IOT
- [x] Supports async, multithreads
- [x] Rust is safe
- [x] Strong tiping
- [x] Statically tipada
- [x] The compiler is able to infer types in some cases
- [x] Immutable variables by default
